# Platform-Specific Notes:
# - Apple Silicon: Default PyTorch installation includes MPS support
# - NVIDIA GPU: Use torch with CUDA 11.8 support  
# - CPU only: Default installation sufficient

# Installation Instructions:
# 1. Create virtual environment:
#    python -m venv venv
#    source venv/bin/activate  # Linux/Mac
#    venv\Scripts\activate     # Windows
#
# 2. Install PyTorch with appropriate backend:
#    # Apple Silicon (recommended)
#    pip install torch torchvision torchaudio
#    
#    # NVIDIA GPU (CUDA 11.8)
#    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
#    
#    # CPU only
#    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
#
# 3. Install remaining requirements:
#    pip install -r requirements.txt
#
# 4. Download required datasets:
#    python data/download_datasets.py
